## API Used

API : https://api.coingecko.com/api/v3/simple/price  

Query Parameters :  
- ids = bitcoin, ethereum, cardano, solana, ripple, polkadot, dogecoin, tron, litecoin, chainlink, avalanche-2, polygon, stellar, cosmos, monero, uniswap, vechain, tezos, filecoin, theta-token  
- vs_currencies = usd  

Full API URL :  
https://api.coingecko.com/api/v3/simple/price?ids=bitcoin,ethereum,cardano,solana,ripple,polkadot,dogecoin,tron,litecoin,chainlink,avalanche-2,polygon,stellar,cosmos,monero,uniswap,vechain,tezos,filecoin,theta-token&vs_currencies=usd  

Description :  
this api is used to fetch real-time cryptocurrency prices in usd for multiple coins
